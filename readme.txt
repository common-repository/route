== Description ==

Now marketing automation is really for Marketers, not Devs!

Without coding, collect data from your website forms, track website events and 
integrate several 3rd party applications to start campaigns for lead nurturing 
and customer engagement within minutes, not months.


== Installation ==

1. Extract 'route.zip' to your '/wp-content/plugins/' directory.
2. In the admin panel under plugins, activate Route.
3. Add your Organization Id from Route on the settings page.
